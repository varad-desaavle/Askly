# 🚀 Askly – AI Chat Application

A full-stack AI chatbot built using the MERN stack that allows users to create multiple chat conversations and receive real-time AI-generated responses using the Groq API.

## ✨ Features

- 💬 Multiple chat threads
- 🤖 AI-powered responses
- 💾 Persistent chat history
- ⚡ Fast REST APIs
- 📱 Responsive user interface

## 🛠️ Tech Stack

**Frontend**
- React.js
- Vite
- CSS3
- Axios

**Backend**
- Node.js
- Express.js
- MongoDB
- Mongoose
- Groq API

## 📦 Installation

```bash
# Clone the repository
git clone https://github.com/varad-desaavle/Askly.git

# Backend
cd backend
npm install
npm run dev

# Frontend
cd frontend
npm install
npm run dev
```

## 🔑 Environment Variables

Create a `.env` file inside the `backend` folder.

```env
MONGODB_URI=your_mongodb_uri
GROQ_API_KEY=your_groq_api_key
PORT=5000
```
